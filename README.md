# TODO App 

A beginner-level project to learn React.

#### How to run:
1. Clone/download the repository.
2. Open the terminal and `cd` into the project folder.
3. Run `npm install` and `npm start`.

The app will run at [http://localhost:3000](http://localhost:3000) in your browser.
